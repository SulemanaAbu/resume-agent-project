# Resume Analysis and Career Recommendation Agent System

## 📌 Description
This project implements a multi-agent system using SPADE to analyze user skills and recommend career paths.

## ⚙️ Requirements
- Python 3.10+
- SPADE library

## 📦 Installation
pip install spade

## ▶️ Running the System

1. Start XMPP server:
spade run

2. In another terminal:
python run_agents.py

## 🤖 Agents
- ResumeAgent: Extracts skills
- AnalysisAgent: Processes data
- RecommendationAgent: Generates recommendations

## 📊 Output
Displays:
- Detected skills
- Recommended careers
- Missing skills
